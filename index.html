LSTM1: Stock Market Prediction Using Deep Learning

This project is about predicting stock prices using deep learning. I used models like LSTM, RNN, and BiLSTM. I collected stock data, trained models, tested predictions, and built a simple recommendation system.

What This Project Does

The project gets stock data from Yahoo Finance. Then it tries different models to see which one predicts stock prices the best. After that, it gives suggestions based on the predictions.

What I Did

  • I made a script to get stock prices using the Yahoo Finance API.
  • I tested three models (LSTM, RNN, BiLSTM) on Apple (AAPL) stock.
  • I saw that the RNN model gave the best results. It followed the trend well.
  • I finished the LSTM model and used it for recommendations.
  • I improved the backtesting system to check if the predictions were working or not.
